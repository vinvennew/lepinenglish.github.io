乐拼英语（LePinEnglish）项目 README 文档
项目简介
乐拼英语（LePinEnglish）是一款专注于英语自然拼读学习的网页应用，主打“学得快用得好”的核心定位，适配英语学习者（尤其适配青少年/儿童），通过智能监测、趣味交互的方式，帮助用户夯实单词基础、提升拼读能力，支持单词管理、进度追踪等功能，操作简洁直观，兼顾实用性与趣味性。
应用采用本地数据存储，无需服务器部署，打开网页即可使用，核心聚焦自然拼读训练，同时支持单词导入导出、错题复习等个性化功能，助力用户高效掌握英语单词。
项目核心信息
- 项目名称：乐拼英语（LePinEnglish）
- 项目定位：自然拼读专项学习网页应用
- 目标用户：英语学习者（适配青少年/儿童，支持家长陪同学习）
- 技术依赖：HTML、CSS、JavaScript、Tailwind CSS（样式框架）、Noto Sans SC 字体
- 数据存储：本地 localStorage 存储，数据仅保存在当前设备，安全便捷
- 核心目标：通过自然拼读训练、智能监测和错题复习，帮助用户快速掌握单词，提升英语学习效率
核心功能概况
1. 引导学习频道（默认首页）
核心作用：提供自然拼读专项训练，支持正常学习和错题抽测两种模式，帮助用户通过趣味交互掌握单词拼读技巧。
- 拼读交互：通过趣味操作引导用户掌握自然拼读逻辑，验证单词拼写正确性。
- 辅助功能：提供发音、音标、提示等支持，降低学习难度，提升学习效率。
- 智能监测：自动记录学习表现，智能标记单词掌握状态，为后续复习提供依据。
- 反馈机制：及时给予学习反馈，增强学习趣味性和成就感，激发学习动力。
2. 词库管理
核心作用：实现单词/短语的全流程管理，支持课本单词导入导出，满足个性化词库需求。
- 单词操作：支持单词的新增与删除，灵活拓展词库内容。
- 导入导出：实现词库的备份与迁移，方便导入课本单词，适配学习需求。
- 词库展示：清晰呈现所有单词及学习状态，便于快速查看和管理。
3. 进度追踪
核心作用：直观呈现学习数据，帮助用户及家长掌握学习进度，便于针对性开展复习。
- 数据总览：清晰展示单词掌握情况分布，快速了解整体学习进度。
- 错题专区：汇总薄弱单词，支持重点复习，精准提升学习效果。
- 单词状态明细：详细呈现每个单词的掌握状态，便于精准定位薄弱点。
4. 其他辅助功能
- 数据重置：方便用户重新开始学习，灵活管理学习数据。
- 界面交互：提供便捷的页面切换和适配功能，提升使用体验。
- 提示说明：引导用户正确操作，降低使用门槛，保障学习顺利进行。
环境要求
运行环境
- 浏览器：Chrome、Edge、Firefox 等主流浏览器（版本≥80）
- 网络：首次打开需联网加载 Tailwind CSS 框架和 Noto Sans SC 字体，后续可离线使用（依赖本地存储）
- 设备：适配电脑、平板等设备，支持响应式显示
快速使用说明
1. 打开网页文件，自动加载初始示例单词（本地存储无数据时）；
2. 默认进入“引导学习频道”，点击“开始学习新词”进入正常学习模式，或点击“进入滚动抽测”复习错题；
3. 学习过程中可通过拖拽字母组合或手动输入完成拼写，提交后系统自动标记单词状态；
4. 进入“词库管理”可导入课本单词、新增/删除单词，或导出词库备份；
5. 通过“进度追踪”查看学习数据，针对不熟练和忘记的单词进行重点复习。
注意事项
- 数据仅存储在当前设备的浏览器本地，清除浏览器缓存或更换设备会导致数据丢失，建议定期导出词库备份；
- 导入单词时需确保JSON格式正确，且包含英文、中文、字母组合（数组格式）等必填字段，否则会跳过无效数据；
- 自然拼读字母组合需与英文单词拼接一致，若不一致会提示确认，避免输入错误影响学习效果；
- 发音功能依赖浏览器 SpeechSynthesis 接口，部分浏览器可能存在发音差异，建议使用主流浏览器。
